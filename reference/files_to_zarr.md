# Convert a native files Daf to Zarr format

Convert a native files Daf to Zarr format

## Usage

``` r
files_to_zarr(files_path, zarr_path)
```

## Arguments

- files_path:

  Path to the source native files Daf directory

- zarr_path:

  Path to the destination `.daf.zarr` (created)

## Value

NULL, invisibly. Called for its side effect.
