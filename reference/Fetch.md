# Fetch (Deprecated)

This function has been removed in DataAxesFormats v0.2.0. Use
[`LookupVector`](https://tanaylab.github.io/dafrjulia/reference/LookupVector.md)
with
[`AsAxis()`](https://tanaylab.github.io/dafrjulia/reference/AsAxis.md)
instead.

## Usage

``` r
Fetch(property, ...)
```

## Arguments

- property:

  A property to fetch.

- ...:

  Additional arguments (ignored).

## Value

Does not return; always raises an error.
